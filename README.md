## ✨ About

**NexusMP Launcher** is a PC launcher project made for private SA-MP/open.mp servers.

The project is currently in early development, so features and direction may change over time.

**NexusMP Launcher** คือโปรเจกต์ Launcher PC สำหรับเซิร์ฟเวอร์ส่วนตัวของ SA-MP/open.mp

โปรเจกต์ยังอยู่ในช่วงเริ่มต้น ฟีเจอร์และแนวทางการพัฒนาอาจเปลี่ยนแปลงได้ในอนาคต
